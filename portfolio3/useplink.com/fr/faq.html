Questions à
propos de Plink

Simple, efficace, et pratique.

Combien coûte Plink ?

Les fonctionnalités de base de Plink sont gratuites. Vous payez uniquement des frais de transaction sur les paiements qui ont réussi. Vous pouvez trouver ces frais sur notre site tarifs Mollie.

Certaines fonctionnalités sont payantes. Vous trouverez une liste de ces tarifs sur notre site Mollie.

Comment s’inscrire?

Plink est directement connecté à votre compte Mollie. Si vous n’avez pas encore de compte, vous pouvez ouvrir un compte Mollie ici. Attention : Mollie est seulement disponible pour les entreprises, vous ne pouvez pas activer un compte si vous êtes un particulier.

Comment fonctionnent les liens de paiement?

Entrez simplement un montant et une référence pour générer un lien de paiement. Vous pouvez ensuite envoyer ces liens à vos clients par email, WhatsApp, ou par un autre service de messagerie.

Une fois que votre client a cliqué sur le lien, il sera dirigé vers l’écran de paiement où il pourra effectuer son paiement. Une fois payé, l’argent sera versé directement sur votre compte Mollie.

Qui est derrière Plink ?

Plink est une application créée par Mollie et fonctionne avec Mollie Connect, ce qui vous permet de vous connecter à Plink directement depuis votre compte Mollie.

Quels modes de paiement peuvent être utilisés ?

Plink permet à vos clients de payer par carte de crédit, PayPal, Bancontact, iDEAL, ainsi que par tous les autres modes de paiement disponible chez Mollie.

Comment fonctionnent les emails de paiement?

Téléchargez votre facture et entrez l’adresse email de votre client où le lien de paiement doit être envoyé. Nous nous occupons du reste et vous pouvez suivre en temps réel le statut de votre paiement.

Y a t-il une API Plink?

Nous souhaitons d’abord étendre l’offre de Plink avec d’autres fonctionnalités de base avant de développer une fonctionnalité plus avancée comme une API. Si l’API Mollie couvre vos besoins actuels, nous vous recommandons d’implémenter cette solution pour le moment.
